# Work Task Prompts

用途：保存当前由主窗口确认、可交给执行窗口直接执行的最终提示词。

# 当前状态：搜索 / 筛选第六轮执行提示词

本轮执行目标：

- 待办筛选只提供清单和重要性；待办搜索增加标题 / 标题加详情及模糊 / 精准匹配。
- 待办搜索初始条件继承已应用待办筛选，但搜索条件变化不回写筛选。
- 逐字符搜索即时更新待办卡片；清空关键词恢复待办筛选结果。
- 待办筛选选择“不限”或点击“重置”时即时恢复结果并取消筛选按钮高亮。
- 日界面和待办界面切换时分别恢复各自搜索文字、清空键和筛选高亮。
- 不修改数据库、独立待办看板和待办卡片既有交互规则。

验收方式：

1. 全量 Python 语法解析。
2. PyQt offscreen 检查待办精简弹窗、选项回填、实时恢复和动态清空键。
3. 主窗口离屏检查筛选继承、搜索隔离、日 / 待办关键词恢复和筛选高亮。
4. 人工 GUI 检查待办卡片结果、空结果提示和弹窗布局。
