# 当前状态：搜索 / 筛选第四轮执行提示词

本轮执行目标：

- 周界面搜索和筛选增加“时间范围”。
- 时间范围使用周一、周二、周三、周四、周五、周六、周日、全部八个同排圆点。

验收方式：

1. 全量 Python 语法解析。
2. PyQt offscreen 检查八点选择联动、选项回填与弹窗渲染。
3. 周窗口离屏检查筛选继承、搜索隔离、动态清空键和清空恢复。
