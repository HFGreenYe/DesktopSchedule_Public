# 当前状态：搜索 / 筛选第三轮执行提示词

本轮执行目标：

- 日课表允许打开既有搜索设置和筛选弹窗。
- 复用卡片模式查询状态与纯查询服务，不复制匹配算法。
- 当前日期与跨午夜所需次日源先构成课表内存快照，再应用筛选和关键词。
- 逐字符输入、条件变化和清空关键词同步更新课表日程块与 DDL 线段。
- 无匹配结果时继续显示时间网格。
- 不改课表颜色、拖拽、拉伸、详情与右键交互。

验收方式：

1. `py_compile` 检查目标文件。
2. PyQt offscreen 检查课表源过滤、逐字符刷新和清空恢复。
3. 检查卡片 / 课表切换时结果一致。
4. 人工 GUI 检查课表块、DDL 线段和空结果网格。
