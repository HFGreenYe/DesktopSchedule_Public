# 当前状态：搜索 / 筛选第一轮执行提示词

本轮执行目标：

- 在日界面卡片模式中，点击搜索框左侧放大镜后显示一个独立“搜索 / 筛选”弹窗。
- 弹窗显示关键词、范围、清单、状态、重要性、日程类型等选项设置。
- 当前只实现入口和面板壳，不实现真实搜索 / 筛选生效。
- 非日界面或课表模式点击时给出轻提示，不打开面板。
- 切换页面、进入添加页或切到课表模式时关闭面板。

验收方式：

1. `py_compile` 检查目标文件。
2. PyQt offscreen 构造新弹窗，确认默认选项和关闭流程正常。
3. 人工 GUI 检查放大镜点击、位置和样式。
