# Work Task Prompts

用途：保存当前由主窗口确认、可交给执行窗口直接执行的最终提示词。

---

# 当前状态：搜索 / 筛选第二轮执行提示词

本轮执行目标：

- 日界面卡片模式的筛选范围固定为当前日期。
- 筛选条件包括清单、重要性、状态和时间形式。
- 搜索条件在筛选有效时复制筛选，在普通界面时默认不限；搜索中修改条件不回写筛选。
- 搜索范围支持标题、标题加详情；精准为字段完整相等，模糊为字段包含关键词。
- 搜索框逐字符刷新，尾部 `×` 仅在非空时显示；清空后恢复原筛选或普通显示。
- 长关键词在失焦后使用左侧省略形式显示。
- 当前旧 `ScheduleSearchPanel` 保留给后续“全部日程”，本轮不实现全局结果页。

验收方式：

1. `py_compile` 检查目标文件。
2. 纯查询规则检查精准 / 模糊、范围、清单、状态、重要性和时间形式。
3. PyQt offscreen 检查动态 `×`、弹窗默认值、筛选快照继承和清空恢复。
4. 人工 GUI 检查面板样式、逐字符刷新和长文本省略。
