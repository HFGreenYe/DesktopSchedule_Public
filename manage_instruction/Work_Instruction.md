# Work Instruction

用途：本文件用于“当前阶段”的执行指令发布。
- 仅保留正在执行或即将执行的任务。
- 已完成阶段请归档到 `History_Instruction.md`，避免本文件长期堆积。

## 当前状态

当前无进行中的施工任务，等待下一阶段指令。

## 执行日志硬性要求

执行窗口每轮完成后必须更新 `manage_instruction/Work_Log.md`，至少记录：
1. 本轮任务名称。
2. 实际修改文件清单。
3. 验证命令与结果摘要（或人工验证结论）。
4. 未完成事项。
5. 风险、疑点或需要决策窗口确认的问题。

若执行失败，也必须记录：
1. 失败位置。
2. 关键报错摘要。
3. 是否已回滚，以及当前工作区状态。
