# Work Formulation

用途：记录当前阶段的总体规划、目标边界和小工单路线。由主窗口在审核决策窗口方案后维护。

# 当前状态：搜索 / 筛选功能第二轮

当前小工单：日界面卡片模式真实搜索与筛选。

目标：

- 日界面查询范围固定为当前日期；周、月后续分别固定为当前周和当前自然月。
- 筛选状态与临时搜索状态独立保存，清空关键词后恢复搜索前的筛选结果。
- 未应用筛选时，搜索的清单、重要性、状态和时间形式初始均为不限；已应用筛选时，搜索初始条件复制当前筛选，但搜索中的修改不回写筛选。
- 搜索范围支持标题、标题加详情；精准搜索要求字段完整相等，模糊搜索要求字段包含关键词。
- 搜索框逐字符更新结果，文本非空时显示尾部清空键，长文本失焦后从左侧省略。

已归档阶段：

- 月界面功能补齐阶段。
- 月日程持久 panel 详情与编辑路由阶段（`MP-0 ~ MP-5`）。
- 坐标看板与主题选色弹窗第一版。
- 坐标看板归档后增强与高 DPI / 多屏回归。
- 日 / 周 / 月持久弹窗课程表模式阶段。

---

## 当前边界

- 本轮仅让日界面卡片模式真实生效；日课表、周界面、月界面和待办暂不接入。
- 查询基于当前日期已加载日程的内存快照，逐字符输入不重复扫描数据库。
- 不修改数据库结构，不保存搜索关键词和筛选条件到 `QSettings`。
- 第一轮 `ScheduleSearchPanel` 暂不作为日界面入口，保留给后续“更多 → 全部日程”全局结果窗口复用。
- “全部日程”仍需先设计结果承载页面，本轮不伪实现全局搜索。
