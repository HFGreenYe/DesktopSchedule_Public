# 当前状态：搜索 / 筛选功能第六轮

当前小工单：将搜索与筛选接入待办列表。

目标：

- 待办列表保留独立于日程的筛选基线和临时搜索状态。
- 搜索与筛选分别保存独立条件；搜索清空后恢复已应用筛选。
- 待办筛选只包含清单和重要性；待办搜索额外包含标题 / 标题加详情和模糊 / 精准匹配。
- 共享主窗口搜索框按日界面与待办界面分别保存和恢复关键词。
- 待办筛选存在实际约束时持续高亮工具栏筛选按钮，重置后取消高亮。

已归档阶段：

- 月界面功能补齐阶段。
- 月日程持久 panel 详情与编辑路由阶段（`MP-0 ~ MP-5`）。
- 坐标看板与主题选色弹窗第一版。
- 坐标看板归档后增强与高 DPI / 多屏回归。
- 日 / 周 / 月持久弹窗课程表模式阶段。
- 日界面卡片与课表搜索 / 筛选阶段。
- 周界面卡片与课表搜索 / 筛选阶段。
- 月界面当前自然月搜索 / 筛选阶段。
