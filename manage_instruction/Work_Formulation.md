# 当前状态：搜索 / 筛选功能第一轮

当前小工单：日界面卡片模式查询面板入口。

目标：

- 点击主窗口日界面搜索框左侧放大镜后，在界面左侧显示“搜索 / 筛选”选项面板。
- 第一轮只做面板壳和入口，不让查询条件真实影响日程列表。
- 面板先承载关键词、范围、清单、状态、重要性和日程类型选项，为下一轮查询模型接线预留。

已归档阶段：

- 月界面功能补齐阶段。
- 月日程持久 panel 详情与编辑路由阶段（`MP-0 ~ MP-5`）。

- 坐标看板归档后增强与高 DPI / 多屏回归。
- 日 / 周 / 月持久弹窗课程表模式阶段。

归档位置：

- `manage_instruction/History_Log.md`

## 当前边界

- 仅作用于日界面卡片模式；课表模式、周界面、月界面和待办暂不接入。
- 本轮不修改数据库，不新增持久化配置，不实现真实查询过滤。
- 点击“应用”只给出占位反馈；下一轮再把条件接入 `ScheduleQueryService` 和卡片刷新。
- 切换出日界面、进入添加页或切到课表模式时隐藏面板，避免悬浮窗状态残留。
- 不顺手实现导出、换肤、加密或云同步。
