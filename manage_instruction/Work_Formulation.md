# 当前状态：ReportLab PDF 真实导出与分页预览已完成

当前小工单已完成 PDF 保存路径、ReportLab 文件生成、导出弹窗视觉选项接线、真实分页预览和 PNG 固定画布分页多图，Markdown 行为保持不变。PNG 运行烟测因当前会话项目解释器提权额度受限仍待本地复核。

完成目标：

1. PDF 已使用系统保存对话框选择路径，并自动保证 `.pdf` 后缀。
2. PDF 与 Markdown 复用同一 `ExportPayload`，内容类型、某日 / 某周 / 某月 / 全部范围真实参与过滤。
3. 纯色 / 渐变背景、标题 / 详情 / 备注字体与颜色、加粗和斜体均由当前弹窗选项驱动。
4. ReportLab 已支持中文字体嵌入、可搜索文本、普通卡片整卡换页和超长详情续页。
5. 成功、失败和取消保存继续复用现有明确反馈语义。
6. PDF 设置变化使用 `300ms` 防抖并在单线程池中串行生成临时文件；内嵌预览显示第一页和总页数，放大预览使用 `QPdfView` 多页显示真实 A4 页框、分页、页码和最终字体排版。

当前边界：

- 不修改数据库字段、Repository 查询语义和日程 / 待办写入逻辑。
- PDF 尺寸固定为 A4；弹窗尺寸选项继续只服务 PNG。
- 默认图片 / 自定义背景、导出历史、默认目录持久化和导出后自动打开仍不实现。
- 临时 PDF 只服务预览，旧任务结果不加载，旧文件在文档解除后自动重试清理；真实保存与预览共享 PDF 写入锁，避免并发注册字体。
- “默认”背景入口已接入 9 个内置选项：8 组程序化图案和 `assets/Output_Background/GreenLeaf.jpg`；一次显示 5 张竖向卡片，滚轮可双向循环浏览并点击选中。默认使用第 1 组，选择结果会进入真实 PDF / PNG 背景；GreenLeaf 在 A4 或 PNG 目标比例中居中等比铺满，手动裁剪焦点和选择持久化仍待后续接入。
- PNG 已按固定画布分页多图方案接线：比例预设映射真实像素，自定义尺寸限制为 `320–8192px` 且单张不超过 4000 万像素；内容放不下时生成 `_001 / _002…` 编号图片，预览和保存共用参数化 ReportLab 分页布局并由 Qt PDF 精确栅格化。纯色、渐变、程序化背景和 GreenLeaf 均进入 PNG，图片背景首版使用居中等比铺满。
- 下一轮先完成 PNG 本地烟测复核，再决定是否进入 GreenLeaf 手动裁剪焦点和背景选择持久化；开始前重新建立阶段合同。
