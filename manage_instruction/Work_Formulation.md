# 当前状态：日界面右键收起 / 放下已完成

本轮在日界面共享右键菜单增加窗口收放状态，卡片模式和课表模式共用同一实现；现有顶部挂起按钮、独立挂起窗口及周 / 月界面逻辑保持不变。

完成目标：

1. 日界面展开时右键菜单显示 `setup.svg + 收起`，收起后头部右键菜单在同一位置显示 `setdown.svg + 放下`。
2. 收起后保留完整 `HeaderBar` 以及搜索 / 按钮组到正文分界线之间的原留白；高度按搜索框和五个工具按钮的最大实际下沿加 `20px` 动态计算，不依赖固定窗口高度。正文隐藏但不销毁，底部两角沿用主窗口现有圆角。
   Header 在主窗口纵向布局中固定为顶部对齐，避免正文隐藏后 Qt 将剩余空间平均分配到头部上下而使全部控件下移。
3. 收起绘制继续使用收起前完整窗口高度作为渐变源，下沿颜色保持展开界面原坐标色；放下恢复收起前的窗口高度、尺寸约束、布局间距、卡片 / 课表模式和原本可见的详情弹窗，不额外刷新日程数据。
4. 收起状态切换周、月、待办或进入添加、排序、筛选等正文功能前自动放下，避免隐藏正文污染其他路由。
5. 收起状态仍可使用原挂起按钮；从挂起窗口恢复后继续保持收起状态，两套功能互不替代。

当前边界：

- 收放入口只在日界面启用，周 / 月 / 待办共享菜单不增加该动作。
- 不修改头部控件布局、天气行为、卡片 / 课表内部布局及数据刷新合同。
- 日期点击后的未来天气弹窗属于后续独立增强，本轮不提前实现。
- 项目 `.venv` 全量语法检查、日页面卡片 / 课表总体功能回归、右键与收放回归以及 PDF / PNG 共享面板回归均已通过，未发现功能被改写或破坏。
